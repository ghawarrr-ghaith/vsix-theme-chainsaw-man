# Chainsaw Man Theme

"I love bad guys!"

## Themes
- **The Hero of Hell (Denji):** Rusty orange, blood red, and dirty devil hunter whites.
- **Control (Makima):** Pale pinks, hypnotic yellows, and dominant blacks.
- **Blood Fiend (Power):** Fiend blood red, jacket blue, and chaotic energy.
- **Fox (Aki):** Cool slate blues, mournful greys, and KON.
