# Chainsaw Man Theme

A collection of VS Code themes inspired by the characters from Chainsaw Man.

## Available Themes

*   **CSM: The Hero of Hell (Denji)** - Inspired by Denji's Chainsaw form with vibrant orange and dark industrial tones.
*   **CSM: Control (Makima)** - Inspired by Makima's elegant yet intimidating aesthetic with red and cream colors.
*   **CSM: Blood Fiend (Power)** - Inspired by Power's energetic and chaotic nature with pink and blood-red tones.
*   **CSM: Fox (Aki)** - Inspired by Aki Hayakawa and his contract with the Fox Devil, featuring sleek blue and dark grey styles.

## Installation

1.  Open Chrome or your browser of choice.
2.  Download the `.vsix` file.
3.  Open VS Code.
4.  Go to Extensions -> "..." -> Install from VSIX...
5.  Select the downloaded file.
6.  Go to File -> Preferences -> Color Theme and select one of the Chainsaw Man themes.
